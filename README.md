# Análisis de la libertad en *El Paraíso Perdido* de John Milton

Presentación web interactiva, en formato **PechaKucha 20×20** (20 diapositivas, 20 segundos cada una), sobre el concepto de **libertad** en *El Paraíso Perdido* de John Milton.

## Contenido

- **`index.html`** — Presentación completa, autocontenida (HTML + CSS + JavaScript, sin dependencias externas). Cada diapositiva incluye el concepto, una cita real del texto y el desarrollo de la idea, de modo que se sostiene por sí misma.
- **`Guion_PechaKucha_Libertad_ParaisoPerdido.md`** — Guion detallado del que parte la presentación.
- **`propmt.yaml`** — Especificación original del encargo.
- **`tools/`** — Utilidades de conversión EPUB/PDF → Markdown usadas para extraer la fuente.

## Uso

Abre `index.html` en cualquier navegador moderno.

| Acción | Tecla / control |
|---|---|
| Navegar | `←` `→`, clic en los laterales, o los puntos inferiores |
| Autoavance PechaKucha (20 s) | `espacio` o el botón **Auto 20s** |
| Índice de diapositivas | `O` |
| Imagen sugerida | `I` |
| Pantalla completa | `F` |

## Despliegue

Sitio estático desplegado en **Netlify** (ver `netlify.toml`). No requiere paso de build: publica la raíz del repositorio.
